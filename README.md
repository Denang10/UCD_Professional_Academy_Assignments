# UCD_Professional_Academy_Assignments
This repository showcases assignments completed as part of the University College Dublin (UCD) Professional Academy courses. It demonstrates practical skills in data analysis, machine learning &amp; Python programming, including code implementations, notebooks, and reports.

Each individual folder pertains to one of the five courses completed as part of this programme. Each folder contains a description of the course, the final report presented as well as the accompanying work. Finally, the certificate indicating the final grade is also displayed.
